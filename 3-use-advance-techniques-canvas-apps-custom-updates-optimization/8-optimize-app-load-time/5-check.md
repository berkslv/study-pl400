### 3. You used the ClearCollect function to preload the data from a table into a collection. However, only 500 of the 1,000 available rows in the table were loaded. What could be causing the problem?

- **Review the app setting Data row limit value.** ✓
- Use the ClearCollect function from OnStart instead of OnVisible.
- Perform a separate call to Clear followed by a call to Collect.
- Use a Dataverse table view that includes all 1,000 records.
