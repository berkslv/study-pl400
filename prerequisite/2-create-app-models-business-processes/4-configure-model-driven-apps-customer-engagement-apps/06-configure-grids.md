# Configure grids

The Power Apps grid control allows users to view, open, and edit records from views and subgrids.

## Grid Capabilities

- **Infinite scrolling** - browse data indefinitely until finding the desired records
- **Customizable column appearance** - adjust the visual appearance of columns for specific needs

## Adding a Grid Control

To add the grid control to views for a table:

1. Select **Solutions** from the Power Apps maker portal
2. Select your solution (e.g., Common Data Services Default Solution)
3. Inside the solution, navigate to **Objects** and select **Apps**
4. Select your model-driven app to open it in the app editor
5. From the **Pages** panel on the left, select the table name to expand it
6. Select the form you want to edit — hover to reveal the **Edit** pencil icon (or use the ellipsis)
7. In the form editor canvas, select the area where you wish to place a grid control
8. Select the **Components** menu from the left navigation rail (or **Component** from the command bar)
9. Expand the **Grid** option (or search for "grid")
10. Select **Editable Grid**

## Configuring the Editable Grid

In the **Add Editable Grid** pop-up pane:
- Select the table you want to link to this control
- Choose the **View** or **Layout**
- Select which device this component appears on: **Web**, **Mobile**, and/or **Tablet**
- Select **Done** when satisfied

After adding the grid to the form canvas:
- A **Properties** pane appears on the right with numerous customizable options
- Select **Save and publish** to record the changes

> **Note**: If you don't have a model-driven app in your solution, you can access the Forms editor directly from your list of Dataverse Tables.
