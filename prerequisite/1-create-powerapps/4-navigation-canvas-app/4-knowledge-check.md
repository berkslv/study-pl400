## Knowledge check for navigation in a canvas app in Power Apps

### Question 1

In Power Apps, the `Back` function sends users to what screen?

- A hard-coded screen.
- A predefined screen.
- **The previous screen that the users moved from.** ✓
- The home screen.

### Question 2

Which is a valid screen transition in Power Apps when you're using the `Navigate` function?

- `ScreenTransition.Jump`
- **`ScreenTransition.Cover`** ✓
- `ScreenTransition.Erase`
- `ScreenTransition.FadeOut`

### Question 3

In certain scenarios, you might create screens but not build navigation to these screens. What's the purpose of this choice?

- To give other editors or owners insight into how the app was built in case more changes are needed later.
- To create documentation about the app.
- To keep track of certain controls, settings, formatting, and colors used in your app so they're easy to find and copy.
- **All are reasons to have hidden screens.** ✓
