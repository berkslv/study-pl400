# Configure charts overview

Charts provide an interactive view of data within model-driven apps.

## Creating or Configuring a Chart

1. From the Power Apps maker portal, select **Tables**
2. Select the table where you wish to create the chart (ideally one that already has data)
3. From the **Tables** dashboard for your selected table, look in the **Data experiences** pane and select **Charts**
4. From the list of charts, select an existing chart or choose **+ New chart** from the command bar
5. Either option opens a chart editor in a **separate browser tab**

## Chart Editor Options

In the chart editor, you can:

- **Name the chart** and pick the **type of chart** to display
- Under **Legend Entries (series)**: select the data to graph and how to itemize it:
  - Average
  - Count: All
  - Count: Non-empty
  - Max
  - Min
  - Sum
- Under **Horizontal (Category) Axis Labels**: select a complementary field for your data
- **Preview**: if your table has data, you can see a live preview as you select Legend Entries and Horizontal Axis Labels
- Optionally add a **description** for your chart

6. When satisfied, select **Save & Close** to close the chart editor and return to the charts list

## Using Charts

Once created, charts can be:
- Displayed in your model-driven app
- Added to dashboards
