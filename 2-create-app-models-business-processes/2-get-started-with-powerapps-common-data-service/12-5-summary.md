# Summary

- 3 minutes

Congratulations on using Microsoft Dataverse and creating your first tables.

Microsoft Dataverse helps you securely store and manage data used by business applications. This data is accessible across Microsoft Power Platform. Both standard and custom tables in Dataverse offer a secure, cloud-based storage solution.

## What you've learned

- Microsoft Dataverse stores data in tables.
- Whenever possible, use default tables to simplify maintenance and reduce customization effort.
- You can create new tables and define relationships between them.
- You can add data to a table from the **Tables** screen, either manually or by importing from various supported sources.
