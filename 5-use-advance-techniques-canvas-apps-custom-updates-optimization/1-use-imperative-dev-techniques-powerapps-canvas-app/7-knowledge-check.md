# Module assessment

This unit contains a module assessment to test your knowledge of the topics covered in this module.

> **Note:** This assessment is interactive and can be taken at [Microsoft Learn](https://learn.microsoft.com/en-us/training/modules/use-imperative-dev-techniques-powerapps-canvas-app/7-knowledge-check). Sign in and answer all questions correctly to earn a pass designation on your profile.

## Topics covered

The assessment tests your understanding of:

- Imperative vs. declarative development in Power Apps
- The three types of variables in Power Apps (global variables, context variables, and collections)
- When to use each type of variable
- How variables work in Power Apps (temporary, session-scoped, no auto-initialization needed)
- Self-referencing variables and storing records in variables
