# Summary

In this module, you learned about:

- What environments are in Microsoft Dataverse.
- Creating and managing environments with Dataverse.
- Adding Dataverse to an existing environment.
- Adding users and roles within an environment.
- Creating a custom security role.
- Monitoring/managing settings for an environment.

In summary, Dataverse environments provide you with a highly customizable array of tools to create/modify environments, add user access, and use security roles to match a wide variety of business requirements.
