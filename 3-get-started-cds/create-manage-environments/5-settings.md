# Manage Settings in an Environment

You can monitor the usage and service health of your environments by going to the Power Platform admin center.

## Steps to Monitor an Environment

1. Sign in to the [Power Platform admin center](https://admin.powerplatform.microsoft.com/).
2. Select **Environments** in the left-hand pane.
3. Select an environment that has Dataverse in it.
4. The environmental dashboard displays information under several panes:
   - **Details** pane: Select **See all** to see more information and update portions of it.
   - Other panes show the current version of Dataverse, what Power Apps are in the environment, and more.

The Power Platform admin center allows you to quickly access and check on any of your environments incorporating Dataverse.
