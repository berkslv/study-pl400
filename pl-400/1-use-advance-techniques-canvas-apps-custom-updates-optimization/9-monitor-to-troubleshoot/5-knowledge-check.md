# Module assessment

## Answer the following questions to see what you've learned.

### Question 1

How do you start the Monitor tool?

- Monitor is always running when you are in Power Apps Studio.
- **By selecting the Advanced tools tab in Power Apps Studio.** ✓
- Launch it from the make.powerapps.com link.
- Install it from Microsoft AppSource.

### Question 2

You want to run Monitor on a published app. What do you need to do?

- Launch Monitor and invite the user.
- Enable the Debug published app setting and then launch Monitor and invite the user.
- **Enable the Debug published app setting, save and publish the app, and then launch Monitor and invite the user.** ✓

### Question 3

Monitor can't be used to identify which one of the following problems?

- **Accessibility issues** ✓
- Delegation issues
- Slow queries of data
- Errors from Dataverse data source actions
