# Check your knowledge

> 6 minutes

Answer the following questions to see what you've learned.

---

**1. What is required for your app to send telemetry to Application Insights?**

- New apps are preconfigured to send telemetry to Application Insights.
- ✅ **Configure the Instrumentation Key, and then save and publish the app.**
- Use the Application Insights connector.
- Install Application Insights from Microsoft AppSource.

---

**2. How do you send a custom message from your app to Application Insights?**

- Use the `Notify()` function.
- Use the `Patch(CustomMessage, { Message: "Your message" })` function.
- ✅ **Use the `Trace()` function.**

---

**3. You can use default app telemetry that is sent to Application Insights to analyze which one of the following issues?**

- Accessibility issues
- Delegation problems
- ✅ **Slow screen load time**
- Errors from Dataverse data source actions
