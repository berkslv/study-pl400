# Module Assessment — Work with relational data in a Power Apps canvas app

- 6 minutes

## Knowledge Check Questions

**1. What are commonly used functions in Power Apps to connect related data sources not in Microsoft Dataverse for apps?**

- Parent and Child
- Relate and LookUp
- Filter and Search
- **✅ Filter and LookUp**

---

**2. What is important to remember when using a LookUp function to query for additional information?**

- **✅ Each LookUp function creates a new connection to the data source, which you should evaluate to avoid performance concerns.**
- LookUp automatically caches data calls to reduce round trips.
- LookUp isn't used to query additional information.
- LookUp only works after configuring the Filter settings.

---

**3. Dataverse has built-in relationship functionality. To take advantage of it, you need to do which of the following?**

- Nothing, Dataverse automatically determines your data relationships.
- Dataverse isn't used for relational data.
- **✅ Create the relationship for the table in the Data section of Power Apps Studio.**
- Create the relationship at the tenant level for Microsoft Power Platform.
