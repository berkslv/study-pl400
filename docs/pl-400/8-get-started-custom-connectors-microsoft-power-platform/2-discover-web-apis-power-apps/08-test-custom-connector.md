# Exercise - Test the custom connector

- 5 minutes

In the last exercise you created a custom connector for your Power Apps app. Now, you will test the connector through this exercise.

> **Note:** Power Apps requires either a Microsoft 365 license or a free trial.

## Test the custom connector

1. To test whether your custom connector works as expected or not, go to the **Custom Connectors** menu on the left-hand side and click the edit (pencil) button on the right-hand side.
2. Click the **4. Test** tab, and you will see the connector has an active connection.
3. Choose the first operation on the left-hand side and click the **Test operation** button, and you will see the result that the connector works as expected.
4. If for whatever reason the web API stops working you will see an error screen. If you see this kind of error screen, get your pro-dev team to help fix the error.
