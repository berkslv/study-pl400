# Test the custom connector

- 5 minutes

Maria now has a custom connector for a web API to use in her Power App application. But before she can use it, she needs to test it to make sure it runs as expected.

## Testing the custom connector

The last step of creating a custom connector is to check whether it works as expected. Power Apps Studio gives you the ability to test your connector using a built-in test tab.

During this testing stage, you can test the connection and the API operations. Power Apps provides example test results, both passing and failing, of the custom connector.

Once you complete the test, you are now good to go build a Power Apps app with the custom connector.
