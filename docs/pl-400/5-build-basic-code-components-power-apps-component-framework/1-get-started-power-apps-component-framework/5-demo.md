# Demo of the Power Apps code component

This unit walks through a video demonstration of creating a Power Apps code component.

The demo covers the end-to-end process of:

1. Initializing a new code component project using the Power Platform CLI (`pac pcf init`).
2. Implementing the component behavior and styling using TypeScript and CSS.
3. Testing the component locally using the test harness (`npm start`).
4. Packaging the component as part of a Dataverse solution and deploying it.

> **Note:** Refer to the video in the official Microsoft Learn module for the full walkthrough:
> [Demo of the Power Apps code component](https://learn.microsoft.com/en-us/training/modules/get-started-power-apps-component-framework/demo)
