## Answer the following questions to see what you learned.

**1. Where do you configure and customize your app?**

- In Power Apps Mobile
- In the Power Apps admin center
- In Microsoft Dynamics 365
- **Power Apps home page (make.powerapps.com)** ✓

**2. What type of data can you use in an app?**

- Local data sources exclusively
- Dataverse and SQL are the only data sources you can use with Power Apps
- **Power Apps can use multiple sources of data, as long as your data sources have valid connections** ✓
- Only Excel files on OneDrive

**3. Before you create an app, what kinds of things should you consider?**

- What you want your users to experience when they use the app
- What the business requirements are
- What you need for a data model
