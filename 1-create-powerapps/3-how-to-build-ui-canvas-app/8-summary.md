# Summary

- 3 minutes

In this module, you explored how to design the user interface of a Microsoft Power Apps canvas app.

## Key topics covered

- **Themes** - Apply consistent styling across your app using built-in theme options.
- **Branding** - Incorporate custom colors and fonts using named formulas.
- **Icons** - Add and customize Icon controls to enhance UI functionality.
- **Images** - Use the Image control to display local files or data source images.
- **Personalization** - Use the `User()` function to personalize apps based on user identity.
- **Form factors** - Configure apps for Phone or Tablet and preview how your app looks on various devices.
